Skill-Format

Skills werden automatisch injiziert wenn die Beschreibung mit der Anfrage übereinstimmt (Keyword-Match ≥ 12%).

Name: Eindeutiger Slug (wird Verzeichnisname)
Beschreibung: Keywords die eine Aktivierung auslösen
Module: Leer = alle Agenten können den Skill bekommen

Max. 2 Skills werden pro Anfrage injiziert.
