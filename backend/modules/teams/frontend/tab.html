Einrichtung

Um Kumio in Teams nutzen zu können, musst du einen Azure Bot (Bot Framework) registrieren.

So funktioniert die Einrichtung:

  1. Gehe ins Azure Portal und erstelle eine Ressource vom Typ Azure Bot.
  2. Hinterlege als Messaging-Endpunkt (Webhook) zwingend eine externe HTTPS-URL, die auf deinen Kumio-Server zeigt: https://deinedomain.com/api/teams/messages
  3. Notiere dir im Azure Portal die Microsoft App ID und generiere ein Client Secret (App Password).
  4. Wechsle in Kumio zu Einstellungen → Module, aktiviere Microsoft Teams und trage dort App ID und Password ein.
  5. Gehe abschließend im Azure Portal bei deinem Bot unter Kanäle, füge Microsoft Teams hinzu und speichere. Fertig!
